GP Virtual System

This GP monitoring system tracks health metrics using Temperature (DS18B20 Sensor), Heart rate (SEN0203 Sensor), and a blood pressure monitor integrated via Node-RED, Tasmota, and Bluetooth. A secure SQL database, accessed through a PHP website, ensures only registered patients and doctors can view data. The project helps patients monitor their health from home and receive free prescriptions conveniently.
